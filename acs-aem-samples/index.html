What ACS AEM Samples is!

A collection of well commented AEM code samples of usual building blocks.

What ACS AEM Samples isn't!

An application package that should be installed.

How to use ACS AEM Samples!

Read and learn from the code and copy/paste classes as baseline for your own implementations.

Samples

Filters

Executed in a chain against every HTTP request, Filters provide a powerful hook into the HTTP Request processing context.

MBeans

Interact with your code via JMX and the Apache Felix MBean Web UI.

Query Builder

AEM's Querybuilder provides an extensible abstraction for querying content in AEM.

OSGi Services

One of the most powerful abstractions in AEM! OSGi services a the perfect place to develop business logic for re-use across all your AEM applications.

Servlets

HTTP Servlets. GET. POST. 'nuff said.

Sling Adapter Factories

Create your own .adaptTo(..) implementations.

Sling Resources

Sling tooling to mimic and extend Sling Resource abstractions.

Sling Resource Providers

Expose data not in the JCR as if it were in the JCR!

Sling Scheduled Services

Run code on a given schedule or interval.

Workflow

Workflow provides a powerful toolset for applying business logic to content in AEM.
