What is ACS AEM Tools?

A set of tools for making AEM 5.6 developer lives easier.

How do I use ACS AEM Tools?

ACS AEM Tools is ONLY for use on development and QA environments. Download the latest release as a CQ Package and install via Package Manager.

How to use ACS AEM Commons!

Add as a sub package from repo.adobe.com using Maven or manually deploy the package using package manager.
