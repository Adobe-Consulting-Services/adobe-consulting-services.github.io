What ACS AEM Commons is!

A way to bootstra AEM project with common functionality, a set of reusable components, and an AEM development toolkit.

What ACS AEM Commons isn't!

A framework, library with required 3rd party dependencies, or a library that changes standard AEM functionality by default.

How to use ACS AEM Commons!

Add as a sub package from repo.adobe.com using Maven or manually deploy the package using package manager.
